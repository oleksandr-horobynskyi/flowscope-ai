# FlowScope AI: Pricing & Savings Model

## 1. Вартість розробки MVP

Формула:

`10 людей x 7 днів x 3 години на день x $10/год = $2,100`

Розрахунок:

- `10 x 7 x 3 x 10 = 2,100`

Отже, орієнтовна вартість створення MVP FlowScope AI:

- **$2,100**

## 2. Модель тарифів

Пропонується 4 категорії користування:

### 1. Free
- Ціна: **$0 / місяць**
- Для кого: тестування продукту, перший дотик
- Що входить:
- базовий доступ до 1 сценарію
- обмежена кількість запитів
- базовий AI-analysis
- без збереження розширеної історії

### 2. Basic
- Ціна: **$10 / місяць**
- Для кого: very small business / solo founder
- Що входить:
- регулярне використання бота
- базове ранжування кандидатів
- summaries і shortlist
- базові screening questions
- 1-2 активні вакансії або невеликий обсяг задач

### 3. Pro
- Ціна: **$15 / місяць**
- Для кого: малий бізнес із регулярним наймом або потоком запитів
- Що входить:
- все з Basic
- більше лімітів по сценаріях і запитах
- transparent scoring
- skills gap analysis
- краща логіка пріоритезації
- кілька одночасних вакансій / напрямів

### 4. VIP
- Ціна: **$25 / місяць**
- Для кого: бізнеси, які хочуть максимум користі та персоналізації
- Що входить:
- все з Pro
- highest limits / майже без обмежень
- пріоритетна обробка
- розширені сценарії
- більш глибокий аналіз
- персоналізований workflow / premium support / early access до нових функцій

## 3. Логіка відмінностей між тарифами

Ключова різниця між планами може будуватись на 4 параметрах:

1. **Кількість запитів**
- Free: мінімально
- Basic: для регулярного базового використання
- Pro: для активного бізнес-користування
- VIP: максимальні ліміти

2. **Глибина аналізу**
- Free: базова відповідь
- Basic: базовий structured output
- Pro: scoring + skills gap + better prioritization
- VIP: розширений аналіз і персоналізація

3. **Кількість сценаріїв**
- Free: 1 демо-сценарій
- Basic: recruitment MVP
- Pro: recruitment + додаткові business scenarios
- VIP: повний доступ до всіх актуальних сценаріїв

4. **Рівень сервісу**
- Free: self-service
- Basic: стандартний доступ
- Pro: розширені можливості
- VIP: priority / premium support

## 4. Економія для клієнта

Базове припущення:

- вартість 1 години роботи власника / менеджера = **$10/год**

### Мінімальний сценарій економії

FlowScope AI економить хоча б:

- **2 години на день**

Розрахунок:

- `2 години x $10 = $20 економії на день`

Якщо рахувати 22 робочі дні на місяць:

- `22 x $20 = $440 економії на місяць`

Отже, навіть у консервативному сценарії:

- **економія клієнта = близько $440 / місяць**

### Сильніший сценарій економії

Якщо продукт знімає навантаження на рівні 1 людини на місяць:

- `1 full-time employee x 160 годин x $10 = $1,600 / місяць`

Отже, верхня межа простої оцінки економії:

- **до $1,600 / місяць**

## 5. ROI для клієнта по тарифах

### Basic — $10 / місяць
- економія: `$440 - $1,600 / місяць`
- чиста вигода: `$430 - $1,590 / місяць`

### Pro — $15 / місяць
- економія: `$440 - $1,600 / місяць`
- чиста вигода: `$425 - $1,585 / місяць`

### VIP — $25 / місяць
- економія: `$440 - $1,600 / місяць`
- чиста вигода: `$415 - $1,575 / місяць`

## 6. Окупність розробки для проєкту

Вартість MVP:

- **$2,100**

Скільки платних клієнтів потрібно, щоб покрити цю суму за 1 місяць:

### Якщо всі на Basic
- `$2,100 / $10 = 210 клієнтів`

### Якщо всі на Pro
- `$2,100 / $15 = 140 клієнтів`

### Якщо всі на VIP
- `$2,100 / $25 = 84 клієнти`

### Змішаний сценарій

Наприклад:

- `50 Basic x $10 = $500`
- `40 Pro x $15 = $600`
- `40 VIP x $25 = $1,000`

Разом:

- **$2,100 / місяць**

У такому сценарії MVP окупається приблизно за:

- **1 місяць**

## 7. Короткий висновок

FlowScope AI має дуже зрозумілу economics модель:

- MVP коштував близько **$2,100**
- користування може коштувати **$0 / $10 / $15 / $25**
- навіть у базовому сценарії клієнт може економити близько **$440 на місяць**
- у сильнішому сценарії економія може сягати **$1,600 на місяць**

Тобто навіть платний тариф у `$10–25` виглядає дуже доступним на фоні потенційної щомісячної економії часу й операційного навантаження.
