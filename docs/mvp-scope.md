# MVP Scope

## MVP Goal

Show how FlowScope AI can accept a raw business problem, recognize it as a recruitment case, and route the user into a practical CV-screening workflow.

## Included in MVP

- landing page with product explanation and CTA
- Telegram bot as the main interaction channel
- raw business request intake
- clarification questions from the assistant
- task classification
- routing into the HR / recruitment branch
- vacancy / candidate requirement intake
- recruitment screening demo logic
- structured final output

## Structured Output Includes

- candidate ranking
- summaries
- transparent scoring logic
- shortlist
- screening questions
- next-step recommendation

## Not Included in MVP

- full ATS platform
- job board integrations
- video interviews
- voice or facial analysis
- multi-branch fully implemented platform
- full custom admin dashboard
- production-grade analytics panel

## Why This Scope Makes Sense

The team chose a narrow but highly visible scenario because it is:

- easy to explain in a demo
- painful enough to matter for SMBs
- structured enough to evaluate
- broad enough to demonstrate the future platform logic
